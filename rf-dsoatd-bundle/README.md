# RF-DSOATD Daily Sheet — bundle

Contents:
- RF-DSOATD Daily Sheet.html  ← open this in a browser
- dsoatd-package/              ← design system (tokens, UI utilities, logo)

## To use
Just open "RF-DSOATD Daily Sheet.html" — it links to the dsoatd-package folder via relative paths, so keep them next to each other.

## Files
- dsoatd-tokens.css   — CSS variables (colors, type, spacing). Includes the new salmon / light-blue / lime accents.
- dsoatd-ui.css       — utility classes (buttons, chips, frames, stat blocks, rows).
- dsoatd-logo.svg     — current logo mark.
- DSOATDLogo.jsx      — React component version of the logo.
- example.html        — quick reference page showing every utility class in use.

## Status color map (used by the sheet)
- Lime   → Paid / Shipped / Cleared
- Blue   → In production / In transit
- Salmon → At risk / Aging
- Hot pink-red → Hold / Needs review
